## 【功能介绍】
**本系统包含管理员、用户两个角色。
管理员角色：个人中心、用户管理、公交线路管理、网站公告管理、友情链接管理、留言管理、系统管理。
用户角色：注册登录、查看公告线路、查看网站公告、留言、我的收藏。**
